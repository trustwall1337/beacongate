Easiest — launch a browser through the proxy:
Spawns a fresh Chrome/Edge window with its own throwaway profile. All traffic in that window routes through the tunnel. Cookies don't leak from your normal browser. Close the window to clean up.
Or — configure proxy manually (Firefox, system-wide, etc.)
Firefox: Preferences → Network Settings → Manual proxy → SOCKS v5 = address above, check "Proxy DNS when using SOCKS v5".
Chrome via CLI: open -na "Google Chrome" --args --proxy-server="socks5://<addr>"
Mac system-wide (replace "Wi-Fi" with your network service name): 
networksetup -setsocksfirewallproxy "Wi-Fi" 127.0.0.1 PORT
# disable when done:
networksetup -setsocksfirewallproxystate "Wi-Fi" off
